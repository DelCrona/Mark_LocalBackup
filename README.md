### 使用方式

在[NGA优化摸鱼体验]**设置面板** -> **插件管理** 导入导出

使用前请务必用本体的导出方式备份数据！

### 配置项

备份文件：点击自动下载标记原始代码

上传标记：本地导入备份文件，导入任何其他文件或内容有问题都会损坏标记功能，只能清除插件缓存重新导入！
